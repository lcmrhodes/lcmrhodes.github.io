Sepsis Flow is one of the strongest examples of the kind of work I want this site to hold: serious research software that still needs interface and systems thinking to become usable.

The project combines multiple prediction services, orchestration logic, and a hosted web interface intended for evaluation workflows. That meant designing for both internal coherence and external clarity.

This work is not a clinical product. As of March 27, 2026, the research is unpublished and not yet peer reviewed, and the tool should not be used for patient-care decisions or clinical deployment.
