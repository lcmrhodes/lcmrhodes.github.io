# lcmrhodes.github.io

Personal portfolio site for Luke Rhodes, built with Astro and deployed to GitHub Pages.

## Stack

- Astro
- Astro content collections
- Plain CSS
- GitHub Actions for GitHub Pages deployment

## Local development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Deployment

Pushes to `main` trigger the GitHub Pages workflow in `.github/workflows/deploy.yml`.
